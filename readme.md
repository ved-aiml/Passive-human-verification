# Passive Human Verification System

An ML-based system to detect human vs bot behavior without using CAPTCHA.

## Features

* Behavioral analysis (mouse, typing, session)
* ML-based classification
* Passive verification (no user interruption)

## Tech Stack

* Python (FastAPI - upcoming)
* Machine Learning (Scikit-learn)
* MySQL (planned)
